## OmniWatch Dashboard

### Setup
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```

### Lint
```bash
npm run lint
```

### Notes
- Time format preference is persisted in localStorage.
- The Sync Stats button simulates a 1.5s API call with a 15% failure rate.
